# Spark++

Spark++ is a Bun workspace for a high-end Gaussian splat engine targeting three.js WebGPU.

## Workspace

- `packages/spark`: reusable library package
- `apps/playground`: Vite + TypeScript playground app

## Getting started

```sh
bun install
bun run dev
```

## Common commands

```sh
bun run build
bun run typecheck
```
